# Controls:
W - Lower the nose of the plane down
S - Lower the nose of the plane up
A - Rotate (Not correct physics, do not touch rn)
D - Rotate (Not correct physics, do not touch rn)

Q - Roll to the left
E - Roll to the right

# How to start:
You can see folder SERVER:
In folder server - run .exe named Flight Simulator 2022
In assets (in unity) find folder Scripts/Network and file in it named ServerBoots, in it change ip 192.168.0.6 by 127.0.0.1, and run your project INSIDE unity
